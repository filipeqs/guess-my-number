# Guess a Number

Guess a number between 1-20

Check the demo hosted on https://filipeqs.github.io/guess-my-number/

---

## Instructions

First clone this repository

```bash
$ git clone https://github.com/filipeqs/guess-my-number.git
```

Open index.html

```bash
open index.html
```

## Features

- Guess a number
- Save highscore
- Start new game
